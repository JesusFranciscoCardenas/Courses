Calculator

This is a basic calculator, please follow instructions to use:

1) Press Calculate button.
2) Capture the first number (if you enter not a number the program ask again until you enter a valid number)
3) Capture the second number (if you enter not a number the program ask again until you enter a valid number)
4) Capture the operation (if you enter not a valid operation the program ask again until you enter a valid operation)
5) See the results
6) To use again repeat steps
